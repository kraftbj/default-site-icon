== Description ==

Sets your main site's site icon as a default for any subsite without one.

Compatible with multinetwork installs, the plugin will check for the current network and use that network's main site's site icon as a default for any subsite that did not specifically set one.

== Installation ==

1. Install the plugin within WordPress via the Plugins->Add New page.
2. Either Network Activate the plugin or activate on an individual subsite.
3. Profit. No options to set, nothing else to do.


== Frequently Asked Questions ==

= What if the main site does not have a site icon? =

A black hole will develop within your server, enveloping the entire world. Or nothing will be returned just as if the plugin didn't exist.
